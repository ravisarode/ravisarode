Featured

EDR agent monitor & detection engineering

End-to-end work on Microsoft Defender for Endpoint: telemetry, rules, and validation—not a toy demo.

What it is

Security research and detection engineering for a major commercial EDR: understanding agent behavior, refining detections, and keeping customers protected without drowning them in noise.

Tech & tools

Microsoft Defender for Endpoint EDR telemetry MITRE ATT&CK Python Threat emulation SIEM-style analysis

Problem solved

  • Noisy or incorrect detections erode trust and slow response—needed rigorous FP/TP analysis and targeted tuning.
  • Customers needed clear, evidence-backed answers when something fired in their environment.
  • Coverage had to be stress-tested against realistic adversary tradecraft, not only happy-path tests.

Impact

  • Improved detection quality through structured analysis of machine events and customer escalations.
  • Reduced analyst toil by tightening suppressions and clarifying which alerts matter.
  • Aligned detection narratives to ATT&CK so security teams can prioritize and communicate risk.

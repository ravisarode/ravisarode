# Resume website

Static HTML/CSS personal resume site.

## Run locally

Open `index.html` in a browser, or from this folder:

```bash
python -m http.server 8080
```

Then visit [http://localhost:8080](http://localhost:8080).

## Pages

- `index.html` — Home
- `work-experience.html`, `education.html`, `project.html`, `technical-skills.html`, `certifications.html`, `volunteer.html`

Styles: `styles.css`
